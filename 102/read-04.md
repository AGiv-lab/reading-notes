##Structuring Web Pages with HTML

*What is HTML and why do we use it?*

**HTML** **Hyper-Text Markup Language** is the standard language used to create and structure content on the web. (the foundation of every web page) HTML can be used to:

-Organize content
-Structure Web pages
-Work with CSS and Javascript
-Display content in browsers

*What are the 3 main parts of an HTML element?*

-Opening tag =<
-Content =p
-Closing tag =</p>

*What is it called when you give an element extra information?*

**Attribute** provides extra information about an HTML element inside an opening tag. <img src="image.jpg" alt="A photo"> src and alt are attributes

*What is a semantic element?*

**An element that clearly describes its meaning to both the browser and to the developers** Thety improve the readability of the code, accessibility for readers, help search engines understand content (SEO), make code easy to understand. 

**example** 
-<header>
-<nav>, _<main>, <section>, <article>,<footer>,<aside>

-<div>, <span>, do not describe the content they contain
