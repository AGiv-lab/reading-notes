##What is **Markdown**? 

**Markdown** is a lightweight markup language that you can use to add formatting elements to plaintext text documents.

###Why do we use Markdown?

*Markdown is the ideal syntax for an easy way to take notes, create content for a website, and produce print-ready documents.*

- What symbol will create a heading in Markdown? #

- How many do you need to create the largest heading? 1

- How many do you need to create the smallest heading? 6

- When making text bold or italicized for emphasis, it is best practice to use which symbol? ** or __

- How do you create a link with Markdown? [NYT]( https:newyorktimes.com)

- What are the three symbols you can use to create an unordered (bulleted) list with Markdown? -,*,+
