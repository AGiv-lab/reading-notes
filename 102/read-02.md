## *What are four important features to look for in a* **text editor**?

### *What do the following commands do*?
1. **pwd** 
- path. shows the folder or path you're in currently

2. **ls**
- list. lists documents in the file 

3. **cd**
- change directory. changes the folder or directory you're in

4. **mkdir**
- Make directory. Makes a folder command to create a folder

5. **touch**
- makes a file
